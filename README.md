# Rumor Detection using PHEME + Knowledge Graph

## Overview

This project implements rumor detection using the PHEME dataset and FakeNewsNet dataset. The system combines machine learning approaches with knowledge graph techniques to improve rumor detection accuracy.

## Project Structure

```
rumor-detection-project/
├── data/
│   ├── raw/           # Raw datasets (PHEME, FakeNewsNet)
│   ├── processed/     # Preprocessed data
│   └── interim/       # Intermediate processing files
├── preprocessing/     # Data preprocessing pipeline
├── knowledge_graph/   # Knowledge graph construction
├── ontology/          # Ontology definitions
├── models/           # Machine learning models
├── notebooks/        # Jupyter notebooks for analysis
├── utils/           # Utility functions
└── config/          # Configuration files
```

## Features

- Multi-dataset rumor detection
- Knowledge graph integration
- Machine learning models
- Preprocessing pipeline
- Ontology-based reasoning

## Installation

1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Download datasets to `data/raw/` directory

## Usage

See notebooks/ for examples and analysis.
